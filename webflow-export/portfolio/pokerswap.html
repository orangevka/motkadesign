PokerSwap

Brand new elegant design for the PokerSwap web-based service. It allows poker players to exchange poker chips between platforms and trade them for cash. The service functions very similar to P2P cryptocurrency trading using an escrow model.

Trading UI

P2P trading usually involves filling out a lot of forms. Users create selling orders and specify payment details. We designed a UI that helps users complete everything correctly. We used grouping, visual hierarchy, and color coding to make these forms simpler and more user-friendly.

Personal Account

The Personal Account page’s UI helps manage selling orders and buying requests. Users can see the current status of each order or request and track the history of all transactions.

Making P2P trading UX better

Both in product UI and on landing page we often educate users on how to trade on the platform. Escrow format is not something every person understands from the get-go. Our visualisations help to understand how service works and start trading.

We establish this trust by making UI easily graspable and predictable, while staying stylish and dynamic. On the example below: buying requests are nested in respective selling orders in which they were originated. That helps to keep track on several trading lines simultaneously.
